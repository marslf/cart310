# CART 310 Final Project

This repository documents my design process for my CART 310 final project.

The project explores a need within a sub-community I belong to, following a user-centered design approach. 
The process includes background research, user research, analysis, design principles, prototyping, and evaluation.

Each folder represents a different stage of the design process, showing the evolution of the project from initial problem space 
to final prototype iterations.

This repository was created to track progress and document the process.
